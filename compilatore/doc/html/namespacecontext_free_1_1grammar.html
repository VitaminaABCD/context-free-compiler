Detailed Description

Contains the class for represent and manipulate a grammar.

USAGE: You can obtain a Grammar instance through the static factory "GrammarFactor" createGrammar; you must pass the quadruple Axiom, Productions, terminals, non-terminals that define a grammar.

es: IGrammar grammar = GrammarFactory.createGrammar(A, P, V, T);

If the grammar is context-free the factory return a ContextFreeGrammar instance ContextFreeGrammar;
